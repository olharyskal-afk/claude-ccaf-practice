# Write Tests Skill

Напиши unit тести для вказаного модуля.

## Instructions
1. Прочитай файл модуля
2. Визнач всі функції які потребують тестів
3. Напиши тести покриваючи: happy path, edge cases, errors
4. Файл тестів назви `*.test.js`
